ls -> lista arquivos
cd -> muda de diretório
pwd -> mostra diretório de trabalho
cat -> mostra conteudo do arquivo de texto
rm -> remove arquivo ou diretório vazio
rm -rf -> remove de forma forçada arquivos ou diretórios de forma recorrente

## FLUXO DE TRABALHO

1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. editar ou criar arquivos
4. git status
5. git add <nome_do_arquivo>
6. git status
7. git commit -m "msg"
8. git status
9. voltar para o passo 3.
